# 5TRAT Core developer documentation

Start with the current project documents:

- [Project and consensus overview](../README.md)
- [Standalone installation](../INSTALL.md)
- [Public network services](../NETWORK.md)

The remaining files in this directory include inherited Bitcoin-family build,
wallet, RPC and protocol documentation. Some still use upstream project names,
binary names or historical compatibility terms. Those references describe code
lineage and internal interfaces. They do not mean 5TRAT shares another
blockchain's ledger, balances, price or peer network.

For public 5TRAT deployment, prefer the container build documented in
[INSTALL.md](../INSTALL.md). Treat older platform notes as developer references
and verify commands against the current source before using them.
